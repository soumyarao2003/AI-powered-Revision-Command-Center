# AI-powered-Revision-Command-Center
Generating a PRD for an AI-powered Revision Command Center for A-Level students
Here's what's included:
Dashboard overview — streak tracker, daily study time, topics completed, and average score at a glance.
Subject tabs — filter the dashboard by subject (Maths, Chemistry, History, English Lit, Biology).
6 AI tools — each button is clickable and fires a real prompt into the chat:

Flashcard generator
Practice quiz (with AI marking)
Concept explainer
Mind map builder
Essay planner
Timetable planner

Weak topics panel — shows topics by mastery score with colour-coded progress bars (red → amber → green).
Upcoming exams — countdown in days with urgency colour coding.
Ask AI bar — type any question and hit Enter or "Ask AI" to get help instantly.
Everything is interactive — try clicking any of the tool cards or typing a question in the bar at the bottom. Want me to build out any specific feature in more depth, like the quiz engine or essay planner?
